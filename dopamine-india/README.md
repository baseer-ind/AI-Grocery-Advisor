# Dopamine India

A simulation-only "app store of apps" for entertainment, stress relief, and
anti-impulse-spending. No real payments, deliveries, or bookings ever occur.

This is a standalone product living inside the AI-Grocery-Advisor repository
as an independent sibling project (separate backend, separate mobile app,
no shared code or database with the grocery advisor).

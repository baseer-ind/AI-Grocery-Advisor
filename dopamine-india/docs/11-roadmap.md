# Roadmap

| Phase | Timeframe | Theme | Key deliverables |
|---|---|---|---|
| Phase 0 | Weeks 1–2 | Foundations | Repo scaffolds, design tokens, auth (Supabase), DB schema + migrations, CI/CD skeleton |
| Phase 1 — MVP | Weeks 3–10 | One sharp category, full loop | Food category end-to-end (browse→cart→checkout→tracking→savings), Profile stats, basic Home dashboard, onboarding |
| Phase 2 | Weeks 11–16 | Category breadth | Shopping, Grocery, Travel, Movies added; Discover search (text+AI); Wishlist/Moodboards |
| Phase 3 | Weeks 17–22 | Social & gamification | Challenges, public profiles, follow graph, achievements, streaks polish |
| Phase 4 | Weeks 23–28 | Remaining categories | Hotels, Beauty, Electronics, Cars, Real Estate, Services, Kids/Pets/Gifts |
| Phase 5 | Weeks 29–32 | Hardening & launch prep | Admin dashboard completion, analytics dashboards, security/store-compliance review, load testing |
| Launch | Week 33+ | GA | Phased rollout (see `15-launch-strategy.md`) |
| Post-launch V2 | — | See `13-v2-features.md` | |
| Post-launch V3 | — | See `14-v3-features.md` | |

See `12-mvp-scope.md` for exact MVP cut line and `18-development-plan.md`
for milestone-level detail with owners/exit criteria.

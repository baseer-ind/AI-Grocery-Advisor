# V2 Features

Builds on MVP once the core loop is validated.

- Remaining high-traffic categories: Grocery, Travel, Movies, Hotels.
- Discover hub: AI search + voice search.
- Wishlist & Moodboards (Dream Room, Dream Wardrobe, Dream Office) with
  AI-assisted generation.
- Social layer: public profiles, follow graph, shareable dream carts.
- Challenges: Luxury, Budget, Room Setup challenges with leaderboards.
- Achievements system + streak mechanics polish (notifications, recovery
  grace period).
- Full Profile stats: calories saved, carbon saved, categories explored
  ring.
- Admin analytics dashboard (funnels, cohorts).
- Push notifications (Firebase) for streak reminders, challenge updates.
- Hindi localization completed across all live categories (MVP ships
  English-first with partial Hindi).
